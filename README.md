# Django Portfolio Website

A personal portfolio website built using Django.

## Features
- Home page with introduction
- Projects section with multiple images per project
- Skills section
- Download CV option
- Responsive (mobile-friendly)
- Admin panel to manage projects

## Tech Stack
- Django
- Python
- HTML, CSS
- SQLite

## How to Run Locally
```bash
git clone https://github.com/Megha-3009/portfolio-django.git
cd portfolio-django
python manage.py runserver
python manage.py migrate
python manage.py createsuperuser
